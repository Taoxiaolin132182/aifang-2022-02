1、先确定抓手原点位置，X方向，Y方向
2、参数项可录入正负号：(+\-)
3、X(单位:mm)：X值越大，抓手的落点就离原点越远
4、Y(单位:mm)：Y值越大，抓手抓的越晚(从抓取时机来看)
5、T(单位:s)：T值越大，抓手抓的越早(减小t参数时，抓取的点变得更前(相当于去提前抓))
6、T参数的换算公式： 目测统计后的Y方向偏差 / 传送带速度
7、传送带速度：在PLC屏幕上的传送带频率 * 比率系数({{ data_1.4 }})
