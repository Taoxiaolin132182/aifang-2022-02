#



1、查看指令的具体用法
    在命令行中输入：  af-ctrl
    根据提示 下一次补全输入

2、查看主程序 和 守护进程 的状态
    af-ctrl help

2、停止主控程序
    af-ctrl stop
    (但要注意：不关闭守护进程的话, 在0-20秒时间内，主控程序会再次被守护进程启动)

3、停止守护进程 和 主控程序
    af-ctrl stop all

4、启动主控程序(重启也是这句)
    af-ctrl start
    (注意：要是主控程序的进程存在，会先停止，再重新启动)
5、启动守护进程(重启也是这句)
    af-ctrl start all
    (注意：要是守护进程存在，会先停止，再重新启动)
    (特别注意！！！：这句指令不会重启主控程序，主控程序不受影响，继续运行)
